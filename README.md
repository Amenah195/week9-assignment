# Product Store App

A simple and modern e-commerce product store built with React.  
This project demonstrates different state management tools in React including Context API, Redux Toolkit, and React Query.

##  Features

- Product listing from API (React Query)
- Search products
- Filter by category
- Shopping cart (Redux Toolkit)
- Increase / decrease quantity
- Remove / clear cart
- Dark / Light mode (Context API + useReducer)
- Grid / List view toggle
- LocalStorage support
- Responsive design
- Toast notifications
- Product details page

##  State Management

- React Query → Fetching and caching products from API  
- Redux Toolkit → Shopping cart management  
- Context API + useReducer → UI settings (theme, view, search, category)

## Tools & Libraries

- React
- Redux Toolkit
- React Query (TanStack Query)
- React Router DOM
- Axios
- Bootstrap 5
- React Toastify
